# Student Management System (Spring Boot)

REST API for managing Students, Books, and Courses using Spring Boot + PostgreSQL.

## Features
- CRUD for Students / Books / Courses
- Assign books to students (One-to-Many)
- Enroll students in courses (Many-to-Many)

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Run
1. Create PostgreSQL database: student_db
2. Update application.properties
3. Run the project
